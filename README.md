# array-merge

Simple library that merges two arrays together. 

```
                 +-----+-----+-----+-----+
                 |  3  |  5  |  2  |  7  |
                 +-----+-----+-----+-----+

                 +   first   +   second  +
                 |   array   |   array   |
                 +-----+-----+-----+-----+
                       |           |
              +--------+           +--------+
              |                             |
              v                             v

        +-----+-----+                 +-----+-----+
step 1  |  3  |  5  |                 |  2  |  7  |
        +-----+-----+                 +-----+-----+

           ^                             ^
           +--------------+--------------+
                          |
           +--------------+
           |
           |
        +--v--+-----+-----+-----+
        |  2  |     |     |     |
        +-----+-----+-----+-----+

        +-----+-----+                 +-----+-----+
step 2  |  3  |  5  |                 |  2  |  7  |
        +-----+-----+                 +-----+-----+

           ^                                   ^
           +--------------+--------------------+
                          |
                 +--------+
                 |
                 |
        +-----+--v--+-----+-----+
        |  2  |  3  |     |     |
        +-----+-----+-----+-----+

        +-----------+                 +-----------+
step 3  |  3  |  5  |                 |  2  |  7  |
        +-----------+                 +-----------+

                 ^                             ^
                 +-----------------------------+
                       |
                       |
                       |
                       |
        +--------------v--------+
        |  2  |  3  |  5  |     |
        +-----------------------+

        +-----------+                 +-----------+
step 4  |  3  |  5  |                 |  2  |  7  |
        +-----------+                 +-----------+

                                               ^
                             +-----------------+
                             |
                             |
                             |
                             |
        +--------------------v--+
        |  2  |  3  |  5  |  7  |
        +-----------------------+
```
